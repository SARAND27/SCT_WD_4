# To-Do Web Application

## SkillCraft Technology Internship - Task 4

This project is a To-Do Web Application developed as part of the SkillCraft Technology Web Development Internship.

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Features

* Add New Tasks
* Mark Tasks as Completed
* Delete Tasks
* User-Friendly Interface
* Responsive Design
* Real-Time Task Management

## How to Use

1. Enter a task in the input field.
2. Click the "Add Task" button.
3. Mark tasks as completed when finished.
4. Delete tasks that are no longer needed.
